1.首先使用镜像CentOS-7-x86_64-Minimal-1611.iso安装一台centos7的服务器。
2.配置好网络，包括DNS，确认能连接到因特网。
3.安装libvirt：
yum install libvirt
4.安装virt-v2v：
yum install virt-v2v
yum update openssl-libs
5.取出resources/virt-v2v放到服务器上，然后替换：
chmod +x virt-v2v
mv -f virt-v2v /usr/bin/
6.取出转换脚本shell/convert.sh放到服务器上，修改权限：
chmod +x convert.sh
mv convert.sh /home/convert.sh
ln -s /home/convert.sh /usr/local/bin/convert
7.生成ssh密钥对，一路回车即可：
ssh-keygen -t rsa
8.安装expect
yum install expect
9.安装cifs
yum install cifs-utils
